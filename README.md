# 腫瘤指引快速查 App

個人本機使用之腫瘤指引查閱、少見癌症防漏、快速參考與本機指引文件庫。

## 功能特色

- 🔍 **全癌別搜尋**：支援中文、英文、同義詞模糊搜尋
- 📄 **本機 PDF 文件庫**：匯入、管理、版本控制指引 PDF
- ✅ **防漏 Checklist**：每癌別可維護必查項目，可勾選、可編輯
- 🎯 **精準化欄位**：疾病狀態、生物標記等選填欄位，留白不阻擋輸出
- 🏥 **健保提醒**：可連結健保可近性資料
- 📋 **快速參考**：一鍵產生參考方向，顯示下一步建議
- 💾 **本機儲存**：所有資料儲存於 IndexedDB，不需後端、不上傳雲端
- 📱 **PWA 支援**：可加入手機主螢幕，支援離線使用

## 使用方式

### 直接開啟（開發模式）

```bash
# 使用任何 HTTP 伺服器
npx serve .
# 或
python3 -m http.server 3000
```

然後開啟 `http://localhost:3000`

### 部署

本應用為純靜態應用，可部署至：

- GitHub Pages
- 任何靜態網頁託管服務
- 本機 HTTP 伺服器

只需上傳整個目錄即可。

## 操作說明

### 1. 初次使用

1. 開啟 App，會看到全癌別分類列表
2. 所有癌別預設為「待維護」狀態
3. 可直接搜尋癌別或開始匯入指引 PDF

### 2. 匯入 PDF

1. 點擊「PDF 文件庫」→「+ 匯入」
2. 選擇已下載的指引 PDF
3. 填寫指引名稱、來源、版本
4. 選擇對應癌別（可多選）
5. 設為目前版本或歸檔

### 3. 編輯癌別

1. 進入癌別詳情頁
2. 點擊「編輯此癌別」
3. 可編輯：
   - 基本資料（中英文名稱、同義詞）
   - 指引路由（主要/次要指引）
   - 必查章節（每行一筆）
   - 外部連結

### 4. 建立 Checklist

1. 進入癌別 →「編輯 Checklist」
2. 新增項目，設定分類與重要性
3. 所有欄位可留白

### 5. 設定精準欄位

1. 進入癌別 →「編輯精準欄位」
2. 可新增單選、多選、文字、數字欄位
3. 所有欄位可留白，不阻擋快速參考

### 6. 快速參考

1. 進入癌別 → 點擊「📋 快速參考」
2. 顯示：
   - 應查指引
   - 本機 PDF 狀態
   - 必查章節
   - 不要漏 checklist
   - 目前資訊不足提醒
   - 下一步建議

### 7. 備份與還原

1. 進入「設定」
2. 「匯出完整備份（含 PDF）」：包含所有資料與 PDF 檔案本體，檔案較大
3. 「匯出備份 JSON（不含 PDF）」：僅含癌別卡、checklist、精準欄位、健保提醒與 PDF 元資料
4. 匯入備份 JSON 還原資料（兩種格式皆可匯入）

> ⚠️ iPhone 請務必將本 App「加入主畫面」使用，否則 Safari 可能在 7 天未使用後
> 清除所有本機資料（含 PDF）。請定期匯出完整備份。

## 資料結構

所有資料儲存於瀏覽器 IndexedDB：

- `cancerCards`：癌別路由卡
- `documents`：PDF 文件元資料
- `checklists`：防漏 checklist 項目
- `precisionFields`：精準化欄位定義
- `nhiNotes`：健保提醒

PDF 檔案本體儲存於獨立的 `OncologyPdfStorage` IndexedDB。

## 技術規格

- **前端**：Vanilla JavaScript（無框架依賴）
- **儲存**：IndexedDB（Dexie 風格 API）
- **PWA**：Service Worker + Manifest
- **樣式**：CSS 變數 + 手機優先響應式設計
- **備份**：JSON 格式

## 注意事項

- 本 App 不是院級醫療資訊系統
- 不串接病人資料、不自動下載指引
- 不輸出「最終治療建議」
- 不上傳資料到雲端
- 僅供個人查閱參考使用
